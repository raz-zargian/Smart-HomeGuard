# Smart-HomeGuard
AWS Workshop: Cloud computing
This is our final project of our studies in Computer Science degree.

This project is related to the field of computer vision and machine learning, specifically
face recognition for smart security systems. The field has evolved from rule-based and
handcrafted feature methods to deep learning approaches, mainly convolutional neural
networks (CNNs). Today, face recognition is widely used in security systems, smartphones,
and IoT applications, with increasing emphasis on personalization and continuous learning
driven by user interaction.
